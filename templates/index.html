Arsitektur VGG16 Transfer Learning

VGG16 adalah arsitektur Convolutional Neural Network (CNN) terkenal yang dikembangkan oleh Visual Geometry Group di Universitas Oxford. Model ini dilatih pada dataset ImageNet yang terdiri atas lebih dari 14 juta gambar dan 1.000 kelas objek.

Dengan menggunakan Transfer Learning (Feature Extraction), kita membekukan 13 lapisan konvolusi bawaan VGG16 untuk mengambil pola-pola dasar (tepi, bayangan, sudut, retakan). Klasifikasi akhir kemudian diserahkan kepada fully connected layer baru yang kita latih khusus pada dataset permukaan jalan.

INPUT -> [224x224x3]
CONV3-64 (2 layers) -> POOL
CONV3-128 (2 layers) -> POOL
CONV3-256 (3 layers) -> POOL
CONV3-512 (3 layers) -> POOL
CONV3-512 (3 layers) -> POOL
FLATTEN -> DENSE (256, ReLU) -> DROPOUT (0.5)
DENSE (1, Sigmoid) -> OUTPUT
